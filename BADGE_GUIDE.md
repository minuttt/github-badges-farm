# Complete GitHub Badges Guide

## Automated Badges (via scripts)

### 1. Pull Shark 🦈
**How to earn:** Merge pull requests
- Bronze: 2 PRs
- Silver: 16 PRs
- Gold: 128 PRs

**Script:** Run `./badge-farmer.sh` - creates and merges PRs automatically

### 2. YOLO 🎯
**How to earn:** Merge a pull request without code review
**Script:** Included in badge-farmer.sh - merges PRs immediately

### 3. Quickdraw ⚡
**How to earn:** Close an issue or PR within 5 minutes of opening
**Script:** Included in badge-farmer.sh - creates and closes issues instantly

### 4. Pair Extraordinaire 👥
**How to earn:** Co-author commits
- Bronze: 1 co-authored commit
- Silver: 10 co-authored commits
- Gold: 24 co-authored commits

**Script:** Included in badge-farmer.sh - uses Co-authored-by trailer

### 5. Heart On Your Sleeve ❤️
**How to earn:** React to content on GitHub with specific emojis
**Manual:** React to issues/PRs/discussions with emoji reactions

## Manual Badges

### 6. Starstruck ⭐
**How to earn:** Get stars on your repositories
- Bronze: 16 stars
- Silver: 128 stars
- Gold: 512 stars

**Strategy:**
1. Create useful/interesting projects
2. Share on social media
3. Contribute to trending topics
4. Add topics to repos for discoverability

### 7. Galaxy Brain 🧠
**How to earn:** Get accepted answers in GitHub Discussions
- Bronze: 2 accepted answers
- Silver: 8 accepted answers
- Gold: 16 accepted answers

**Strategy:**
1. Enable Discussions on this repo
2. Create discussions and mark answers as accepted
3. Participate in other repos' discussions

### 8. Open Sourcerer 🔮
**How to earn:** Have pull requests merged in multiple public repositories
- Need contributions across different repos

**Strategy:**
1. Find "good first issue" labels
2. Fix typos in documentation
3. Add examples to READMEs
4. Report and fix bugs

### 9. GitHub Sponsor 💝
**How to earn:** Sponsor an open source developer/project
**Action:** Go to github.com/sponsors and sponsor someone

### 10. Public Sponsor 💰
**How to earn:** Publicly sponsor someone
**Action:** Sponsor with public visibility enabled
